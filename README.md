This application requires:
-------------

- PHP >= 5.4
- Composer
- Npm

Getting Started
---------------

  - Install vendor using composer

  ``` composer install ```

  - Create database

  ``` create db named 'laravel' on phpMyAdmin and import file 'laravel.sql' file ```

  - Start server

  ``` php artisan serve --port=3000 ```

Enjoy ! (on localhost:3000)
